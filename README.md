# OpsForge

**AI-Powered Army MDMP & Operations Process Tool (Corps to Company)**

OpsForge helps U.S. Army staff and leaders plan, prepare, execute, and assess operations using real doctrine.

### Main Features
- Echelon selector (Corps, Division, Brigade, Battalion, Company) with automatic MDMP or TLP workflow
- Exact MIL-STD-2525D military symbols with correct codes and drawing instructions
- Tactical Overlay tab with MGRS grid and drag-and-drop battlefield geometry (unit symbols, phase lines, engagement areas)
- Upload and analyze unclassified documents (PDF, Word, PPTX, Excel, images, XML, Google formats)
- Generate AR 25-50 formatted orders, annexes, and outputs
- Full "Ask Doctrine" chat with direct, detailed answers and citations from FM 5-0, FM 3-0, FM 1-02.2 and more
- Training Mode with quizzes and flashcards
- On-device AI – completely free and works offline
- Shared workspaces for multi-echelon staff collaboration

### How to Use
1. Open the app link
2. Select your echelon
3. Use MDMP Wizard, Tactical Overlay, or Ask Doctrine
4. Install on your phone home screen for easy access

Built as a Progressive Web App (PWA) – works on iOS, Android, and desktop. Fully offline capable.
